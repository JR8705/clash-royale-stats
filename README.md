# Clash Royale Stats

Aplicação que consulta estatísticas, histórico de partidas e decks
de uma conta do Clash Royale a partir da tag do jogador, consumindo
a API REST oficial do jogo.

## Funcionalidades
- Busca de jogador por tag
- Exibição de estatísticas e decks

## Tecnologias
- JavaScript, HTML, CSS
- API REST do Clash Royale (autenticação por token)

## Como rodar
1. Clone o repositório
2. Crie sua chave em developer.clashroyale.com
3. Substitua `SUA_API_KEY_AQUI` no arquivo X
4. Abra o `index.html` no navegador
